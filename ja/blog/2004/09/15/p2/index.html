VC# でプロパティを自動作成するマクロ

Tweet

Reflection IT Create Property Macro for Visual Studio.Netを試用してみる。

試しにVC#のエディタに

string name

と入力して、選択状態でこのマクロを実行してみると

private string _name;

public string Name
{
	get
	{
		return (this._name);
	}
	set
	{
		this._name = value;
	}
}

と展開された。_name の _(アンダーバー) が気に入らなかったので、マクロのソースのうち Const sPREFIX As String = "_" を "" に変更。

現在では、このマクロにショートカットキーを登録してうはうは生活を送っています。
